# word-count

A small lines/words/characters counting program.

## Getting started

  * Install the latest version of stack, https://docs.haskellstack.org/en/stable/README/ and configure it
  * Build and install word-count by running, stack build, stack install
  * Stack may have to be configured to allow-newer ghc versions
  * Tested with stack version 2.9.3 and stackage snapshot LTS 20.8
